# Family Tree Analysis

Analysis of WikiTree family tree matching results.

### Code

- `analyze_matches.py` — Analyzes matched WikiTree candidates against debt holder records.
- `analyze_non_matched.py` — Examines debt holders who could not be matched to WikiTree profiles.

### Input

- `output/scrape/wikitree/` (WikiTree search results)
- `output/derived/family_tree/` (filtered match data)

### Output

- `output/analysis/family_tree_analysis/`

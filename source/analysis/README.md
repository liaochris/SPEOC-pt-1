# Analysis

Scripts that produce tables, figures, and summary statistics from cleaned data.

## Directory Structure

- `pre1790/` — Pre-1790 certificate distribution, temporal, and geographic analysis
- `post1790_cd/` — Post-1790 continental debt analysis
- `debt_analysis/` — Treasurer matching and Hamilton public debt analysis
- `open_refine_analysis/` — OpenRefine reconciliation and Ancestry results analysis
- `family_tree_analysis/` — WikiTree family tree network analysis

## Output

Generated figures, tables, and CSVs are stored in `output/analysis/{subfolder}/`.

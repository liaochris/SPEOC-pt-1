# OpenRefine Analysis

Analysis of OpenRefine reconciliation results and Ancestry scraper output.

### Code

- `analyze_openrefine_results.py` — Quantitative analysis comparing pre-1790 and post-1790 matched individuals, including holdings concentration, geographic patterns, occupations, and notable biographies.

### Input

- `output/analysis/open_refine_analysis/` (reconciled CSVs)
- `output/derived/post1790_cd/final_data_CD.csv`

### Output

- `output/analysis/open_refine_analysis/` (matched_percentage.png, percent_changes.png)

### Reference

- `reconciliation_analysis.md` — Full report: "Reconciling the Debt: A Quantitative Match of Pre-1790 Certificates and Post-1790 Stock" (David Cho, July 2025).

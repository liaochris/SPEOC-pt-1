# Raw Data

Untouched original data files. Nothing in this directory is generated by code.

## Directory Structure

- `pre1790_cd/` — Pre-1790 continental debt certificates (loan office, liquidated debt, Pierce, marine) as XLSX from National Archives transcriptions
- `post1790_cd/` — Post-1790 continental debt records by state, as XLSX from National Archives T-series records
- `post1790_asd/` — Post-1790 assumed state debt records by state, as XLSX from National Archives
- `census_data/` — 1790 census population data by county and zip code database
- `delegates/` — Constitutional convention (1787) and state ratifying convention delegate rosters
- `shapefiles/` — Historical geographic boundary files for mapping (county and state boundaries, 1790)
- `society_members/` — Society of the Cincinnati membership lists by state

## Correction and Metadata Files

These manually-created CSV/JSON files live alongside the raw data they correct:

### post1790_cd/
- `cd_raw.csv` — Metadata mapping each state's raw XLSX file path, header row, column ranges, and column count for automated import
- `town_fix.csv` — Manual corrections for town name standardization, created by comparing cleaning output against USGS GNIS place name database
- `company_names_fix.csv` — Manual corrections for company/partnership entity names identified during name deduplication
- `occ_correction.csv` — Manual corrections for occupation field standardization across states

### pre1790_cd/
- `corrections.json` — Manual name corrections applied during pre-1790 name cleaning, created by reviewing automated output against original documents
- `manual_corrections.csv` — Additional manual corrections for pre-1790 records identified during quality review
- `cd_info.csv` — Reference metadata for pre-1790 certificate types and their attributes
- `occupations_states.csv` — Occupation data by state for pre-1790 certificate holders

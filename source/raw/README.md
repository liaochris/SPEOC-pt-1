# Raw Data

Untouched original data files. Nothing in this directory is generated by code. Each subdirectory has its own README following the JMSLab raw directory documentation standard.

## Directory Structure

- `pre1790/` — Pre-1790 debt certificates (loan office, liquidated debt, Pierce, marine)
- `post1790_cd/` — Post-1790 continental debt records by state
- `post1790_asd/` — Post-1790 assumed state debt records by state
- `census_data/` — 1790 census population data
- `delegates/` — Convention delegate rosters
- `shapefiles/` — Historical geographic boundary files
- `society_members/` — Society of the Cincinnati membership lists

## File Usage Summary

Files actively used in production code (web app, cleaning pipeline):
- All pre1790/ XLSX files (via web app and cleaning notebooks)
- All post1790_cd/ XLSX files (via cleaning pipeline cd_import_metadata.csv config)
- census_data/: census.csv, countyPopulation.csv, statepop.csv, zip_code_database.xls
- delegates/: both XLSX files (in analysis notebooks)
- shapefiles/: stateshape_1790/, nhgis_state_1790/ (web app mapping)

Files not currently referenced in active code:
- post1790_asd/ XLSX files (no cleaning pipeline exists yet)
- shapefiles/historicalcounties/, shapefiles/stateshape/ (only in archive)

### Overview

This directory provides an assortment of census population and geographical data. 

### Source
The data source is described for each dataset in the description, but is alrgely from historical US Census Bureau data and derived datasets. 

Contact: chrisliao (at) uchicago (dot) edu.

### When/where obtained & original form of files
- All NHGIS files were obtained via IPUMS
- 
All non-csv files were converted into csv's. 

### Description

**`orig/`** — Original data files:
- `county_demographics_1790.csv` — County-level 1790 census data from the NHGIS with population demographic groups (slave, free, total populations with derived percentages and densities).
- `county_pop_fips.csv` — County-level 1790 census data from Social Explorer with population and FIPS data. Includes county names, state abbreviations, and total population.
- `zip_code_database.csv` — ZIP code geographic reference database, used as city-county crosswalk for fuzzy geographic matching. Obtained from unitedstateszipcodes.org's free zip code database

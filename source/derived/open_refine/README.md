### OpenRefine Data

OpenRefine project files and liquidated debt certificate cleaning outputs.

### Contents

- `clean_liquidated_debt_certificates/` — Per-state cleaned CSVs and combined output, plus OpenRefine project archive
- `projects/` — OpenRefine project tarballs (`.openrefine.tar.gz`) for various datasets

### Notes

OpenRefine is a manual reconciliation tool. The project files here preserve the reconciliation state. Whether to keep OpenRefine in the pipeline is a future decision (noted for review).
